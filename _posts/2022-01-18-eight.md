#### Axios   
서버와 데이터를 송수신 할 수 있는 HTTP 비동기 통신 라이브러리   
Promise 객체 형태로 값을 return함   
장점: 응답 시간 설정을 통해 네트워크 에러 발생했을 때 정해진 응답 시간 초과하면 해당 요청 종료시킬 수 있음   
   
##### Axios 설치   
npm install axios --save
